# SnapSphere

## Project Structure
- backend/  (Node + Express + MongoDB)
- frontend/ (React + Vite + Tailwind)

## Setup

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm run dev

## Environment Variables
Copy and create your own env files:

- backend/.env.example -> backend/.env
- frontend/.env.example -> frontend/.env
